| 技术方案                            | 优点                                                         | 缺点                                       |
| ----------------------------------- | ------------------------------------------------------------ | ------------------------------------------ |
| **正则表达式（Rule‑based）**        | 实现简单；延迟极低；可控性强                                 | 规则维护成本高；无法处理口语化表达；泛化弱 |
| **TF‑IDF + 传统分类器（SVM / LR）** | 训练快；实现简单；对小数据集友好                             | 无语义理解；对同义句、口语化表达不敏感     |
| **BERT（预训练 Transformer）**      | 语义理解强；泛化好；可通过微调达到 高精度；可部署在本地或云端 | 推理速度较慢；模型较大；延迟高             |
| **LLM（如 Qwen / Llama）**          | 语义能力最强；零样本/小样本效果好；长尾意图覆盖能力强        | 推理慢；成本高；本地部署困难；延迟高       |
